# 🏋️ Experience Level Prediction System

An interactive Machine Learning web application built with **Python**,
**Streamlit**, and **Scikit-learn** that predicts a user's fitness
experience level based on workout, health, and body measurements.

------------------------------------------------------------------------

# 📌 Project Overview

This application allows users to enter personal fitness information such
as age, gender, BMI, workout type, calories burned, heart rate, body fat
percentage, water intake, and workout frequency. The trained machine
learning model analyzes these features and predicts the user's
experience level.

------------------------------------------------------------------------

# ✨ Features

-   Interactive Streamlit interface
-   Machine Learning prediction
-   Multiple fitness input parameters
-   Data preprocessing using saved transformer
